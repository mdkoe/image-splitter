# Image-splitter

## インストール方法
[Python3](https://www.python.org/downloads/) (3.7.0)
- MacOS
```
brew install python3
```
[ImageMagick](https://www.imagemagick.org/script/download.php)
- MacOS
```
brew update
brew install imagemagick
```

## Usage
コンマンドに以下の通りに実行して下さい
```
python3 main.py filename.zip
```
