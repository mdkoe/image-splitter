# Image-splitter

## インストール方法
[Python](https://www.python.org/downloads/) (3.7.0) 
 MacOS
```
brew install python
```
[Pillow](https://pillow.readthedocs.io/en/latest/index.html) for python3 
- MacOS
```
sudo pip3 install Pillow
```

## Usage
コンマンドに以下の通りに実行して下さい
```
python3 main.py filename.zip
```
